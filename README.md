# hamster-keyboard

一些Hamster的键盘布局配置，大部分来自于热心的网友制作，一些加入了自己的修改。

# [仓输入法（Hamster）](https://github.com/imfuxiao/Hamster)

一款基于「[中州韻輸入法引擎／Rime Input Method Engine](https://github.com/rime/librime)」的 iOS 版本输入法.
